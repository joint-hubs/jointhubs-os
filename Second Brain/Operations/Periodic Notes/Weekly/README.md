# Weekly Notes

Weekly review notes created with `.github/skills/weekly-review/template.md` (copy into your template system).

## Naming Convention

`YYYY-Www.md` (e.g., `2025-W01.md`)

## Typical Content

- Week summary
- Goals achieved vs planned
- Patterns noticed
- Lessons learned
- Next week priorities

## Usage

The Review agent uses these notes for end-of-week synthesis and pattern recognition across weeks.
