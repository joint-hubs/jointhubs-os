# Daily Notes

Daily notes created with `.github/skills/daily-log/template.md` (copy into your template system).

## Naming Convention

`YYYY-MM-DD.md` (e.g., `2025-12-30.md`)

## Typical Content

- Morning intentions
- Tasks for the day
- Energy level tracking
- End-of-day reflection
- Gratitude/wins

## Usage

The Planner and Journal agents read these notes to understand your patterns and help with daily planning.
