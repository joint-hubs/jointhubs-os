# Health

Health and fitness tracking files.

## Files

| File | Purpose |
|------|---------|
| `training-log.md` | Gym sessions with exercises, weights, reps |
| `nutrition-log.md` | Daily macros and meal tracking |
| `food-database.md` | Reference table for food macros |
| `training-progression-plan.md` | Progressive overload targets |

## Nutrition Log Format

Daily summary table:

```markdown
| Date | Day | Type | Protein | Carbs | Fat | kcal | Steps | Rating | Notes |
|------|-----|------|---------|-------|-----|------|-------|--------|-------|
| 2025-12-30 | Mon | Training | 180g | 200g | 70g | 2200 | 10000 | 8/10 | Good day |
```

## Tips

- Log workouts immediately after training
- Track PRs (personal records) in a summary section
- Use consistent food entries for easier tracking
- Link to these from daily notes via `[[training-log]]`
